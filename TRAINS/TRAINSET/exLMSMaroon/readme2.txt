This zip file contains 512 X 512 textures to replace the 1024 X 1024 textures that came with the original download.  You should only install these texture if you are having difficulty using the coaches with their original textures.  Using these textures will downgrade the appearance of the coaches.

Installation Instructions:

1 - Using Windows Explorer, navigate to the installation folder of Train Simulator.  If you installed Train Simulator with the default settings, this will be:

'C:\Program Files\Microsoft Games\Train Simulator'


2 - Once you are into the installation folder, navigate to the 'Trains' folder.  Once in the 'Trains' folder, navigate to the 'TrainSet' folder.

3 - Navigate to the folder called 'exLMSMaroon'.  Copy all of the files from the zip file into this folder.  When asked to overwrite existing files, answer 'Yes'
