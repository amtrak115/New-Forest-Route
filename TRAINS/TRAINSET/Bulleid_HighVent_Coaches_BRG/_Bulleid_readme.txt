Bulleid High Vent Coaches.
BR 1956 Green livery.

This pack contains Bulleid coaches ordered in 1945 and allocated to the LSWR sector.

Included are.....
Bournemouth 6car dining set 292.
3car set 766
2car set 65

Loose BCK
Loose SK
Loose CK

Two BSOs are also included.
These have no set numbers to allow additional sets to be made up.

INSTALLATION
~~~~~~~~~~~~

Unzip to temporary folder.
Drag,Copy or Move the Bulleid_HighVent_Coaches_BRG folder to your Trainset folder. 

You are now ready to create a consist containing this model by using the Activity Editor supplied with Microsoft Train Simulator - the model will not be available in the simulator unless you do this.
WARNING: It is not recommended that you delete these files once installed without deleting the activity and consist files that use them.
