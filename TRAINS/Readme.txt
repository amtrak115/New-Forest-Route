Lydbrook Junction 2 Stock
Class 14xx 0-4-2 tank engine No.1431
Version 1.0 ( February 2011 )

3D Model built by Reg Furniss

-----------------------------------------------------------------------------
This 3D model has been built using TrainSim Modeler

Class 14xx Tank engine No.1431 was used on the auto-trains, and is presented here in GWR war-time black livery
and early BR black livery.

You will need Stuart Williamson`s " 2 cylinder generic sound set " UKTS File ID2910
( add it to your common.sound folder )
You will need Lew Price`s " GWR tanker " CabView file UKTS ID8519 ( add it to 
your common cab folder )

Install instructions:
---------------------
1.    unzip the contents of this .zip to a safe place.

2.    Copy and past the folder  < LJ_14xx > into your Train Simulator/trains/trainset directory.

3.    Make a consist 

4.	Have fun
